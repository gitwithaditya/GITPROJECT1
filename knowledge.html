Knowledge page

Git skills that stick

Beginner friendly

Core ideas

  • A repository is a folder Git is tracking.
  • A commit is a saved snapshot with a message.
  • A branch is a movable line of work.
  • A merge brings changes from one branch into another.

Essential commands

These are the commands you will use again and again.

git status git add . git commit -m "message" git log --oneline

Daily workflow

  • Check your state with git status.
  • Stage finished changes with git add.
  • Commit small, clear pieces of work.
  • Push when you are ready to share.

Good habits

  • Write commit messages that explain the purpose.
  • Pull before starting shared work.
  • Use branches for features and fixes.
  • Review changes before committing them.
